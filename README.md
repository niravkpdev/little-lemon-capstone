# Little Lemon Front-End Capstone Project

This is a React web application for the Little Lemon restaurant. The project includes a homepage and a functional table booking form.

## Features

- Responsive homepage
- Restaurant hero section
- Reserve-a-table form
- Form validation
- Success confirmation message
- Accessible labels for form fields
- Unit tests

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- React Testing Library

## How to Run the Project

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

4. Run the test suite:

```bash
npm test
```
